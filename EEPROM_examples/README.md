Examples for TinyBasRO microcomputer
====================================

The file contains 8 examples in the Tiny BASIC language for the TinyBasRO computer. If you load them into the computer's serial EEPROM memory (you can choose "disk h" or "disk f") you will get 8 examples for working with the microcomputer.

The examples correspond to the examples described in the TinyBasExamples manual for the TinyBasRV microcomputer. Since the TinyBasRO computer has a smaller display (only 16 x 21 characters), the programs are modified to fit on the screen.

In the last example for controlling the traffic light, a total of 3 separate LEDs are used. Red on D2, yellow on D3 and green on D4 via 470 Ohm resistors. In addition, it is possible to connect a chain of 3 multi-colored WS2812 LEDs to D1.

Příklady pro mikropočítač TinyBasRO
===================================

Soubor obsahuje 8 příkladů v jazyku Tiny BASIC pro počítač TinyBasRO. Pokud je nahrajete do sériové paměti EEPROM počítače (můžete si zvolit "disk h" nebo "disk f") získáte 8 ukázek pro práci s mikropočítačem.

Příklady odpovídají ukázkám popsaným v manuálu TinyBasExamples u mikropočítače TinyBasRV. Protože počítač TinyBasRO má menší displej (pouze 16 x 21 znaků), jsou programy modifikovány tak, aby se vešly na obrazovku.

U posledního příkladu pro ovládání semaforu jsou použity celkem 3 samostatné LED. Červená na D2, žlutá na D3 a zelená na D4 přes rezistory 470 Ohm. Navíc je možné připojit na D1 řetězec 3 vícebarevných LED typu WS2812.
